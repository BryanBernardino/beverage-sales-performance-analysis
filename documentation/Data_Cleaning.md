# Data Cleaning

The dataset was prepared using Power Query before visualization.

## Cleaning Process

- Checked for missing values
- Removed duplicate records
- Verified data types
- Standardized product and category names
- Formatted date fields
- Validated numerical columns
- Created calculated columns for reporting

## Data Transformation

Power Query was used to:

- Import the dataset
- Transform data types
- Prepare the data model
- Optimize the dataset for dashboard reporting

These steps improved data quality and ensured accurate dashboard calculations.
