# Data Dictionary

| Column | Description |
|----------|-------------|
| Product | Name of the beverage product |
| Brand | Beverage brand |
| Category | Beverage category |
| Quantity | Number of units sold |
| Unit Price | Selling price per unit |
| Discount | Discount applied to the transaction |
| Total Revenue | Revenue generated from each transaction |
| Order Date | Date of purchase |
| Year | Sales year |
| Region | Customer sales region |

---

## Data Source

The dataset contains historical beverage sales transactions used for sales performance analysis in Power BI.
